# セキュリティとプライバシー

## 個人情報の取り扱いについて

### ✅ 安全な設計

このツールは **完全にブラウザ内で動作** し、個人情報を外部サーバーに送信しません。

### データの保存場所

1. **LocalStorage（ブラウザ内のみ）**
   - 入力データはブラウザのLocalStorageに保存されます
   - データは **お使いのコンピューター内のみ** に保存され、外部には送信されません
   - ブラウザのキャッシュをクリアするとデータは完全に削除されます

2. **生成されたPDF**
   - PDFファイルはブラウザ内で生成され、直接ダウンロードされます
   - サーバーを経由せず、**完全にオフライン** で動作します

### 外部通信

以下のリソースのみ、外部CDNから読み込まれます（初回アクセス時のみ）：

- **PDF-LIB** (https://unpkg.com/pdf-lib@1.17.1) - PDF生成ライブラリ
- **Fontkit** (https://unpkg.com/@pdf-lib/fontkit@1.1.1) - フォント処理ライブラリ

これらは **ツールの動作に必要なライブラリのみ** で、個人データは一切送信されません。

### セキュリティ対策

✅ **実装済み**
- ブラウザ内完結処理（サーバーレス）
- XSS対策（入力値のサニタイゼーション）
- HTTPS通信（GitHub Pagesでホストされている場合）
- LocalStorageの適切な使用
- クライアントサイドのみでのPDF生成

### 推奨事項

#### 🔒 安全に使用するために

1. **信頼できる環境で使用**
   - 個人のコンピューターで使用してください
   - 公共のコンピューター（図書館、ネットカフェなど）での使用は避けてください

2. **使用後のデータ削除**
   - 使用後は必要に応じてブラウザのキャッシュをクリアしてください
   - LocalStorageのデータは手動で削除可能です（開発者ツールから）

3. **ブラウザの選択**
   - 最新版のモダンブラウザを使用してください
   - Chrome、Firefox、Edge、Safariなどを推奨

4. **HTTPSでアクセス**
   - GitHub Pagesなど、HTTPSでホストされているサイトからアクセスしてください
   - ローカル開発環境（localhost）も安全です

#### 📱 データの管理

**自動保存機能について**
- フォーム入力時に自動的にLocalStorageに保存されます
- データを残したくない場合は、使用後に以下の操作を行ってください：

**データ削除方法：**
1. ブラウザの開発者ツールを開く（F12キー）
2. "Application"または"ストレージ"タブを選択
3. "Local Storage"を展開
4. サイトのURLを選択
5. `medicalReceiptData`キーを削除

または、ブラウザのキャッシュをクリアすることで一括削除できます。

### プライバシーポリシー

- **データ収集**: このツールは個人データを収集しません
- **データ送信**: いかなるデータも外部サーバーに送信されません
- **クッキー**: このツールはクッキーを使用しません
- **トラッキング**: アクセス解析やトラッキングは行いません

### よくある質問

**Q: データは安全ですか？**
A: はい。すべての処理はブラウザ内で完結し、データは外部に送信されません。

**Q: インターネット接続が必要ですか？**
A: 初回アクセス時はライブラリの読み込みに必要ですが、一度キャッシュされればオフラインでも動作可能です。

**Q: 京都産業大学がデータを受け取りますか？**
A: いいえ。このツールは京都産業大学とは無関係で、データは一切送信されません。

**Q: LocalStorageのデータは暗号化されていますか？**
A: LocalStorageは平文で保存されます。公共のコンピューターでは使用しないでください。

**Q: スマートフォンでも安全ですか？**
A: はい。同様にブラウザ内で完結するため安全です。ただし、共有端末では使用後にデータを削除してください。
