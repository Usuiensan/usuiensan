# 医療費領収証明書 背景画像について

## 配置方法

1. 京都産業大学の医療費領収証明書をスキャンまたは撮影
2. 画像を `medical-receipt-bg.jpg` という名前で保存
3. このディレクトリ (`assets/img/`) に配置

## 推奨仕様

- **形式**: JPG または PNG
- **解像度**: 300dpi 以上推奨
- **サイズ**: A4サイズ（210mm × 297mm）
- **ファイル名**: `medical-receipt-bg.jpg`

## 注意事項

- 背景画像が存在しない場合は、白紙のPDFが生成されます
- 個人情報が含まれる場合は、事前に削除してください
- 画像は公開リポジトリにコミットしないでください（.gitignore に追加推奨）

## 代替方法

背景画像を使用しない場合、JavaScriptコードが自動的に白紙ベースでPDFを生成します。
